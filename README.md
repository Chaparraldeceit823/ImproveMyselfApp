## 🚀 What this app does

ImproveMyselfApp is an Android app for tracking daily growth in one place. Use it to log habits, workouts, and diet. You can also check your progress with a 4-photo daily tracker and a goal countdown that shows how much time is left.

The app saves your data on the device, so your progress stays there after you close it. It uses a clean Material 3 layout and is built for fast, smooth use.

## ✨ Main features

- Track daily habits in a simple list
- Log workouts for each day
- Record meals and diet notes
- Add 4 photos per day for visual progress
- Use a goal countdown to keep an eye on deadlines
- Keep your data saved on the device
- View a clean dashboard with clear sections
- Use a modern Android interface with large touch targets

## 🧭 How to use the dashboard

The dashboard keeps your daily progress in one place. Use it to move between habit tracking, workout logs, food notes, and photo updates.

### Habit tracking
Add the habits you want to keep each day. This can include water intake, reading, stretching, or any routine you want to build.

### Workout logs
Save your training for the day. Add short notes for sets, reps, cardio, or rest time.

### Diet notes
Record meals, snacks, or meal plans. This helps you see patterns over time.

### Photo tracker
Add up to four photos for each day. Use this to compare your progress week by week or month by month.

### Goal countdown
Set a target date and watch the countdown update. This helps you keep a clear focus on your next goal.
